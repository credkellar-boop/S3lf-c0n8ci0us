# 🧠 S3lf-c0n8ci0us
*An Auto-Cognitive Reasoning Engine*

S3lf-c0n8ci0us is an AI framework that forces LLMs to operate within strict, iterative analytical frameworks (Sol-Plex) before generating output. It utilizes a Rust-native cognitive core to audit, track, and optimize logic streams in real-time.

## Key Architecture
- **Cognitive Core (Rust/PyO3):** Native-speed auditing, performance telemetry, and memory management.
- **Subconscious Lib (JSON):** A modular library of structured problem-solving frameworks.
- **Conscious Surface (Python/Streamlit):** The high-level orchestration interface with live cognitive visualization.

## Quickstart
1. Ensure Rust and Python 3.11+ are installed.
2. `make docker-build` to containerize the environment.
3. `make docker-up` to launch the engine.
4. Input complex engineering problems and observe the live topology of the AI's reasoning path.
